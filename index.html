How To Use

You can use this website to generate sheets of labels from a set of pre-generated images, such as those generated by asset-labelmaker

To use:

  1. Select one of the label sheet templates (contributions welcome if yours isnt listed)
  2. If you are printing on an already-partly-used sheet, set the number of labels to skip (first page only) so you arent printing where a label has already been used (counting left to right, top to bottom)
  3. Upload your images (it can be slow or broken if you try and do more than ~1 sheet at a time, so be aware)
  4. If your labels are smaller and you want to fit more than one per label, use the images per label field to specify how many labels to vertically stack in each label.
  5. If you want to combine full size and half-size labels onto the same printout to use an entire sheet, create another label group with your second set of images. This group can have a different images per label setting than other groups
  6. Click generate and wait for your PDF to generate. You should see your browsers configured PDF handling kick in less than a minute or so
