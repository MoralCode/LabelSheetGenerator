How To Use

You can use this website to generate sheets of labels from a set of pre-generated images, such as those generated by asset-labelmaker

To use:

  1. Select one of the label sheet templates (contributions welcome if yours isnt listed)
  2. If you are printing on an already-partly-used sheet, set the number of labels to skip (first page only) so you arent printing where a label has already been used (counting left to right, top to bottom)
  3. Upload your images (it can be slow or broken if you try and do more than ~1 sheet at a time, so be aware)
  4. If your labels are smaller and you want to fit more than one per label, use the images per label field to specify how many labels to arrange in each label. You can select the arrangement style if you want to have the images tile instead of vertically stacking.
  5. If you want multiple copies of every label in a group, set the quantity field.
  6. If you want to combine multiple types and arrangements of labels (such as full size and half size) onto the same printout to save space and printer passes, create another label group with your second set of images. This group can have a different images and settings than other groups
  7. Click generate and wait for your PDF to generate. You should see your browsers configured PDF handling kick in in a few seconds, although it may take longer if you are providing lots of input images
